Проблема Команда Дорожная карта

1. Проблема и Решение

Проблема

Многие дети растут в цифровой среде, но не всегда получают персонализированное и безопасное образовательное взаимодействие. Родители и воспитатели ограничены во времени, а существующие приложения часто не адаптируются под уникальные потребности ребёнка или не мотивируют его учиться через игру.

Решение

Мы предлагаем интерактивную обучающую игру с элементами адаптивного обучения на базе ИИ: игра автоматически подстраивает задания под уровень ребёнка, использует голосовые подсказки, отслеживает прогресс и даёт рекомендации родителям. Цель — повысить мотивацию, развивать когнитивные навыки и формировать полезные привычки через игровой процесс.

Сфера: Образование, семейные технологии

Ключевой эффект

  • Индивидуальная адаптация учебного плана
  • Повышение мотивации и вовлечённости
  • Сокращение времени учителя на подготовку

4. Дорожная карта и текущий этап

Идея — исследование рынка и гипотез (завершено)
Прототип — интерактивный игровой уровень + простой backend (текущий этап)
MVP — 5 уровней, профиль ребёнка, базовый ИИ для адаптации (1–2 месяца)
Запуск — бета для 100 семей, сбор метрик (1-2 месяца)

Текущий этап

Прототип

Следующие шаги

  1. Сбор тестовой группы и сбор фидбека
  2. Интеграция простого адаптивного движка на базе правил + ML
  3. Подготовка к MVP и тестирование с родителями

5. Как планируем реализовать решение

Ключевые шаги

  1. Дизайн игрового цикла и механик
  2. Разработка фронтенда (React / Vue / простой HTML для прототипа)
  3. Создание backend: хранение профилей, метрик, рекомендаций
  4. Внедрение ИИ: модели ранжирования задач и адаптивной подборки (онлайн fine-tuning необязательно на старте)
  5. Тестирование, безопасность и соответствие нормам для детей

Технологии

  • Frontend: React или чистый HTML/CSS для демо-версии
  • Backend: Node.js / Flask / python
  • База: PostgreSQL / SQLite для прототипа
  • ИИ: PyTorch / TensorFlow (модели для адаптации), embeddings + простая классификация
  • Хостинг: Vercel / GitHub Pages (демо) + Railway/Render для backend

Как используем ИИ

ИИ подбирает сложность задач по реальному времени (точность ответов, скорость), формирует рекомендации родителям и предсказывает зоны, требующие повторения. На этапе MVP мы используем простую модель классификации и правило-бейзлайн, чтобы обеспечить прозрачность.
